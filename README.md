# altis

Altis boxing & fitness brand — website, docs, and digital MVP.

## Structure

- `altis_web/` — Next.js app (deploy this folder on Vercel)
- `docs/` — project documentation and Obsidian vault

## Deploy on Vercel

1. Import this repo on [Vercel](https://vercel.com/new)
2. Set **Root Directory** to `altis_web`
3. Add environment variables from `altis_web/.env.example` (if present)
4. Deploy
