# Altis Website Images — Generation Manifest

## How to use

1. Open `Nano Banana 2 Prompts - Altis Website Images.md` from the `fit` folder
2. Generate each image using the corresponding prompt ID below
3. Save the generated image to the file path shown
4. The website will automatically display it via `next/image`

## Recommended generation order

1. A03 (hero-dark) — most impactful
2. A07–A12 (program images) — 6 images, reuse same prompt style
3. A16 (about-hero)
4. A20 (trial-booking)
5. A26–A31 (gallery) — 6 images
6. A22–A23 (OG images)
7. A25 (footer texture)
8. Remaining section images

## Format

- Use **WebP** format for best performance
- Follow the dimensions above
- Keep file sizes under 200KB per image where possible
